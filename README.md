# 一分钟联盟营销系统

这是一个本地可运行的联盟营销管理原型，围绕“充值配套”计算推广佣金。

## 功能

- 充值配套：配置配套名称、充值金额、一级佣金比例、二级佣金比例。
- 推广员：新增推广员，设置上级推广员，生成专属推广链接。
- 充值订单：选择推广员和配套生成订单，已支付订单自动产生佣金。
- 佣金结算：查看每笔佣金来源，一键结算待结算佣金。
- 数据导出：导出当前浏览器本地数据为 JSON。

## 使用

直接打开 `index.html` 即可使用。数据保存在浏览器 `localStorage` 中，适合演示、内测和梳理业务流程。

后续可以继续扩展：

- 接入真实支付回调。
- 接入后端数据库和登录权限。
- 增加提现申请、风控审核、团队层级报表。
- 支持不同配套的有效期、权益和库存。
