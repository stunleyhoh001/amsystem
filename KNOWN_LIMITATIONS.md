# 已知限制和下一阶段

这个 POS 当前已经可以作为云端多分行测试版使用，但正式大量使用前还有一些需要继续加强的地方。

## 库存并发

当前做法：

- 收款时先扣本机库存。
- 订单和库存立即保存在本机，不等待网络完成。
- 在线时在后台使用 Firestore Transaction 写入销售并扣减云端库存。
- 云端暂时失败时保留本机订单，并加入待同步队列稍后重试。

限制：

- 在线交易已经降低库存覆盖风险。
- 离线期间多设备销售同一库存，恢复网络后系统会保存订单并标记“库存待复核”，仍需要人工核对实际库存。

当前复核流程：

1. 在交易记录筛选“库存待复核”。
2. 点击“前往库存核对”，先刷新云端资料并核对实际数量。
3. 在库存页调整正确数量。
4. 回到订单点击“确认库存已复核”。

下一阶段建议：

- 使用 Cloud Functions 作为唯一收款入口，进一步集中校验权限、库存和订单写入。
- 后期交易量增加后，建立独立的库存盘点与审批流程。

## 离线冲突

当前做法：

- 断网订单先存本机。
- 恢复网络后自动补传订单和库存。

冲突处理：

- 云端事务会先读取订单内所有商品。
- 任一商品库存不足时，不扣减任何商品库存，避免部分成功。
- 订单会退出待同步队列并显示“库存待复核”，避免无限重试。
- 本机自检、设置页、交易筛选、日结和销售 CSV 均会显示待复核状态。

## 权限

当前做法：

- Firebase Google 登录。
- Firestore Rules 限制管理员和授权员工。
- 本机仍保留离线缓存和本地授权资料。
- 联网登录后每 15 分钟复核当前用户授权，恢复网络时立即复核。
- 停用账号或变更进行中班次的授权分行时，会自动退出并保留班次。
- 纯离线密码会话恢复网络后，必须重新使用 Google 验证。
- 管理员离线密码必须先由 Google 管理员在设置页建立，至少 8 位。
- 登录身份使用浏览器会话储存，关闭浏览器即清除；业务资料仍保存在本机。

限制：

- 断网期间无法即时得知管理员刚做的停用操作，只能在恢复联网后生效。
- 授权复核每次只读取当前用户文件；持续开启 24 小时约每台设备 96 次读取，离线时不读取。

下一阶段建议：

- 后期可增加管理员查看各设备最后在线时间和授权核对时间。

## 部署配置

当前做法：

- 使用 `firebase-config.local.js` 保存真实 Firebase 配置。
- 该文件不提交 GitHub。

注意：

- 上线部署时必须确保部署版本能读取真实 Firebase 配置。
- API Key 应限制 HTTP referrers。

## 同步顺序

- 刷新云端前会先补传本机队列，同一时间只执行一轮补传和一轮读取。
- 待同步商品、相关分行库存、分行、员工和设置不会被旧云端资料覆盖。
- 分行、员工和设置属于管理员资料，只有 Google 管理员登录后才会补传。
- 若管理员长期没有登录，后台资料会继续安全保存在本机队列和完整备份中。

## 本机储存

- POS 使用浏览器本机储存支持即时和离线操作，仍受浏览器可用空间限制。
- 核心收款采用整批写入和失败回滚；写入失败时不会改变购物车、订单或库存。
- 无法解析的旧资料会优先另存为 `.corrupt-*` 恢复附件，并包含在完整备份内。
- 建议每日结账后导出一次完整备份，不要使用浏览器“清除网站资料”功能。

## 退款一致性

- 在线退款以 Firestore Transaction 原子处理订单作废和库存回补。
- 已作废订单再次提交不会重复回补库存。
- 离线退款先保存本机并等待新版退款事务，不会用整份商品覆盖云端库存。
- 云端商品缺失时不做部分回补，订单会进入库存复核流程。

## SimplePay 待付款订单

- 未填写 SimplePay 参考号时，POS 会建立待付款订单并预留库存。
- 待付款订单不会计入销售额、班次收入、付款汇总或客户跟进。
- 顾客在 SimplePay 确认付款后，安全云函数会把云端 POS 订单转为正常销售。
- POS 不使用跨项目实时监听；收银员可在收据或交易记录点击“检查付款状态”，每次只读取该订单。
- 顾客未付款时，收银员应作废订单以释放预留库存。
- “取消待付款”会同步取消尚未执行的整合任务；SimplePay 付款前也会反查 POS 状态，已取消订单不能继续付款。
## 蓝牙打印

- 网页蓝牙直连需要 Android 或电脑的 Chrome / Edge，并要求 HTTPS 或 localhost。
- iPhone / iPad 浏览器目前不支持 Web Bluetooth。
- 只支持传统 Bluetooth Classic SPP、不支持 BLE 的热敏打印机不会出现在配对清单中，可改用“系统打印”。
- 不同厂牌 ESC/POS 蓝牙服务可能采用专用 UUID；若能够配对但提示找不到打印通道，需要补充该型号的服务 UUID。
